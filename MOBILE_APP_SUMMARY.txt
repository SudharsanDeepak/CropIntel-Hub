ğŸ†˜ TROUBLESHOOTING:
================================================================================

Problem: "npm not found"
Solution: Install Node.js from https://nodejs.org

Problem: "Android Studio not found"
Solution: Download from https://developer.android.com/studio

Problem: Build fails
Solution:
  cd "AIML Project - App"
  npm install
  npm run build
  npx cap sync

Problem: App crashes on launch
Solution:
  - Check capacitor.config.json exists
  - Verify .env.production has correct URLs
  - Check Android Studio logs (Logcat)

Solution:
  - Rebuild: npm run build && npx cap sync
  - Check console for errors
  - Verify API URLs are correct
